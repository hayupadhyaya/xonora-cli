# Keybindings

## Global

| Key | Action |
|-----|--------|
| `1`–`9` | Switch to tab 1–9 |
| `Space` | Play / Pause |
| `[` | Previous track |
| `]` | Next track |
| `A` | Toggle local audio output on/off |
| `Ctrl+R` | Reconnect (re-runs WebRTC signalling in remote mode) |
| `q` | Quit |

## Navigation (list / table tabs)

| Key | Action |
|-----|--------|
| `j` / `Down` | Move selection down |
| `k` / `Up` | Move selection up |
| `g` | Jump to top |
| `G` | Jump to bottom |
| `Page Down` | Scroll one page down |
| `Page Up` | Scroll one page up |
| `Enter` | Activate / select the highlighted row |
| `Esc` | Cancel / close modal |

## Search

| Key | Action |
|-----|--------|
| `/` | Open search prompt |
| `Enter` | Run search |
| `Esc` | Close search |

## Console

| Key | Action |
|-----|--------|
| `i` | Enter input mode |
| `Enter` | Submit command |
| `Esc` | Exit input mode |
| `Up` / `Down` | Scroll command history |

## Status-bar hints

The bottom status bar shows context-relevant hints per tab. Watch it for tab-specific keys that are not in this global list.
